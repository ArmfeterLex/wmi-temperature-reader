# WMI Temperature Reader

Простая программа для чтения температуры процессора через WMI.

## Возможности

- Чтение температуры CPU из WMI
- Поддержка нескольких пространств имён (root\WMI, root\CIMV2)
- Конвертация в градусы Цельсия
- Обработка ошибок и отсутствующих данных

## Использование

1. Запустите программу
2. Нажмите кнопку
3. Получите температуру

## Встраивание в свои проекты

Код из button1_Click можно скопировать в любой C# проект с Windows Forms и System.Management.

## Требования

- Windows
- .NET Framework
- Поддержка WMI провайдера температур
